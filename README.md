# ONC-MAPPING

## 中文说明

本项目用于支持 **Pink-tailed Worm-lizard（粉尾无足蜥）栖息地调查与制图**，通过外业观测与 UAV 遥感影像结合，完成地表特征提取、地面-影像对齐和栖息地质量评估。

依据文档：
`DOC/Pink-tailed Worm-lizard Habitat Outdoor Investigation (Urumbi Hills) – 10 Oct 2025.md`

### 项目目标

- 核实原生草本植被、微地形与裸岩暴露等关键环境特征。
- 建立地面观测与 UAV 影像对应关系（GCP、尺度参照、地面照片）。
- 优化并验证栖息地质量指标，形成推荐调查点位清单。
- 产出可用于后续分析与汇报的标准化数据与图件。

### 目录结构与用途

- `DOC/`：项目文档目录，存放外业计划、需求说明、用户故事等文本资料。
- `DOC/USR_Story/`：用户故事与方案说明（中英文）文档。
- `COLAB_ED/`：Colab 相关实验脚本与 Notebook（流程验证、原型实验）。
- `DRAFTS/`：草稿代码与临时实验脚本。
- `EXE_VER/`：可执行版本相关源码与构建文件（GUI、流程编排、打包配置等）。

## English Version

This repository supports **Pink-tailed Worm-lizard habitat investigation and mapping** by combining field observations and UAV remote-sensing imagery for feature extraction, ground-to-image alignment, and habitat quality assessment.

Reference document:
`DOC/Pink-tailed Worm-lizard Habitat Outdoor Investigation (Urumbi Hills) – 10 Oct 2025.md`

### Project Objectives

- Confirm key environmental features, including native vegetation, micro-topography, and exposed rock surfaces.
- Build reliable correspondence between ground observations and UAV imagery (GCPs, scale references, field photos).
- Refine and validate habitat quality indicators and generate a recommended survey-site list.
- Produce standardized datasets and map outputs for downstream analysis and reporting.

### Folder Structure and Purpose

- `DOC/`: Project documentation (field plans, requirements, user stories, and notes).
- `DOC/USR_Story/`: User-story and solution documents in Chinese and English.
- `COLAB_ED/`: Colab notebooks and scripts for workflow prototyping and experiments.
- `DRAFTS/`: Draft code and temporary experiment scripts.
- `EXE_VER/`: Source code and build assets for the executable version (GUI, pipeline runners, packaging config).
